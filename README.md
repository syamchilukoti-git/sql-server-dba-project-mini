# SQL Server Backup & Restore Project

  This project demonstrates basic SQL Server DBA operations using T-SQL and SQL Server Management Studio (SSMS).

## Features
  - Database creation
  - Table creation
  - Insert and retrieve data
  - Database backup
  - Database restore
  - Basic SQL queries

## Tools Used
  - Microsoft SQL Server
  - SQL Server Management Studio (SSMS)

## Skills Practiced
  - T-SQL
  - Backup and Restore
  - Database Administration Basics
  - SQL Server Management

## Project Purpose
  This project was created for practicing SQL Server DBA concepts as a fresher.
